# Blog Images Directory

## Image Requirements for SEO

### Featured Images
- **Recommended Size**: 1200x630px (Facebook/Twitter optimal)
- **Format**: JPG or WebP (for best performance)
- **File Size**: Under 200KB (compress for faster loading)
- **Naming**: Use descriptive, SEO-friendly names (e.g., `roofing-website-design-2025.jpg`)

### Image Optimization Tips
1. **Compress images** before uploading (use TinyPNG or similar tools)
2. **Use descriptive alt text** for accessibility and SEO
3. **Include keywords** naturally in image file names
4. **Consider WebP format** for better compression while maintaining quality
